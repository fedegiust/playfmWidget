play.fm Widget jQuery Plugin

Example - With default options

		
			$('#playfmWidget').playfmWidget({type: 'artist', searchItem: 'Richie Hawtin'});


Other options available 
		
			$('#playfmWidget').playfmWidget({type: 'artist', searchItem: 'Richie Hawtin', showPic: true | false, orderBy: "chrono" | "total" | "popindex"});
